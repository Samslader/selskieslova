### 📖 О проекте

**selskieslova** — это открытый словарь народной мудрости, локальных мемов и выражений, которые можно услышать только в самых душевных уголках нашей страны. Здесь собраны слова, которые делают общение проще, ярче и... громче.

---

### 📝 Словарь

| Слово / Выражение | Значение | Пример употребления |
| :--- | :--- | :--- |
| **Газ** | Поехали / Давай быстрее | — Ну что, в город? — **Газ!** |
| **Связь** | До связи / Увидимся | — Ладно, я погнал. — Давай, **связь!** |
| **У меня колом** | У меня встал (в значении застрял/остановился) | Машина в грязи **колом** встала. |
| **Баской** | Красивый / Нарядный | Ох, какой у тебя велик **баской**! |
| **Кумекать** | Общаться / Соображать / Думать | Пойдем **покумекаем** за жизнь. |

---

### 🤝 Как внести вклад?

Если ты знаешь крутое слово, которое мы забыли — не сиди колом! 
1. Форкни репозиторий.
2. Добавь свое слово в таблицу.
3. Создай Pull Request.
